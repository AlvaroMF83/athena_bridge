**`finn_athena_bridge`** es una librería de Python que proporciona una API de DataFrame al estilo de **PySpark** para interactuar con **AWS Athena**. Permite a los desarrolladores y analistas de datos escribir código de transformación de datos declarativo y familiar, que se traduce automáticamente en consultas SQL eficientes ejecutadas en Athena.

El objetivo principal es abstraer la complejidad de escribir SQL manualmente, permitiendo a los usuarios centrarse en la lógica de negocio mientras la librería se encarga de la generación y ejecución de consultas.
